Blurify Privacy Policy

Blurify is a browser extension designed to protect users from sensitive content.

Data Processing

Blurify analyzes Google search queries and images on web pages locally in the browser to detect sensitive content and block keywords.

Data Collection

Blurify does not collect, store, transmit, sell, or share personal data with external servers.

Local Storage

User settings such as blocked keywords and preferences are stored locally using browser storage.

Third Parties

Blurify does not send user data to third parties.

Remote Code

All scripts and the machine learning model are packaged within the extension and run locally.
